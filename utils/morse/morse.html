Morse Code Translator

Convert text into morse code, or vice versa.

Just type normal text to convert to morse code.

Type morse code to convert to normal text.

Use dots (.) and either underscores (_) or dashes (-) for morse letters.

 

Use a single space to separate letters and either a forward slash (/) or two or more spaces to separate words.
